# 暑期课程：中学生学业规划

- 课程目标：通过对目标的分析，和计划的制定，对自我的目标与挑战有更加清晰的定位
- 课程时长：4个课时
- 课程收获
  - 更清晰的目标
  - 学会使用PPT汇报
- 课程对象：谦谦 小逸

## 课程计划
- 1 学业规划介绍
  - 作业：仿照[模版](http://plan.51xuanxiao.com/article-5377)创作一份
- 2 具体分析
  - 作业： 完善
- 3 单独讨论

- 4 PPT汇报成果
